Liste des GET :

GET /beers // Récupère la liste complète des bières
GET /breweries // Récupère toutes les brasseries et leurs informations
GET /breweries/:brewery_id/beers // Affiche les bières par brasserie
GET /users/:user_id/cart // Affiche le panier d'un utilisateur
GET /beers/:beer_id // Récupère les informations détaillées d'une bière spécifique

Liste des POST : 

POST /cart/add // Ajouter une bière au panier
POST /beers/add // Ajouter une bière au catalogue
POST /breweries/add // Ajouter une nouvelle brasserie



Pour héberger le front : Vercel, Nettity

Pour héberger le back : render

Pour la base de donnée : clever.cloud
